# 衡山行

慕着五岳之南岳及距离之短而来。

来之前，做了些许攻略，暗自期待。

开始，乘坐公交，换乘班车，坐看窗外掠过的风景，欣赏并在心中绘画衡山。

下车，跟着导航，随着人流，一路看见旅游带来的发展。购票，正式爬山。

刚开始，柏油路，风景平，不远，右转入石阶小道，明梵音道。梵，佛经用语，梵音，应是颂佛声。遥想古人沿此古道上下山，一路颂者佛音，而古道旁的巨石们，回响着佛音。

途中遇一小湖，名华严，两小瀑布，一不知其名，二曰祥龙吐瑞，其形似口吐水柱，这大概是名之由来。

咦，前路成巨石隘口，底至顶仅比我高少许，若从远处看，似无处可走。

遇到第一个分岔口，大路与小路，择小路行之，直抵南天门。中途购二佛珠、一银剑。

路上无意看到雾中若隐若现的山峰，美。之后到了云层之上，俯瞰云海与山峰，云下与云上，隔了层迷人的云。

最后三个目的地：观日台、会仙桥、祝融峰。观日台为明早观日之地，祝融峰乃山顶。至于会仙桥，太阳落下的地方。兴许是美丽的蝴蝶成双对并于梦境中在会仙桥相会，又或者是众仙人于此奏清乐。

我于会仙桥静静地看着红日从云海之上落下云海。日落时，云海上，孤独的几片云，像是目送太阳，充满不舍，故目光化作迷蒙霞光，为日落失去阳光的天空补充色彩。云、日、山，偶有飞鸟飞过，带来一声清啼，这算得上人间仙境。

夜晚帐篷中露营，黑夜中亮起几点微光与一轮猩红之月，之后又变成皓月高悬。

无眠的夜，不知为何。夜半观星，冷风吹得衣服猎猎作响，通过手机星图软件，我首先找到了星空中最明显的三颗星——它们连成了一道明亮的竖线，从上至下分别为参宿三、二、一。接着，以此为参照物，找到了众多星星，只可惜，无星海，不能看到白带银河。
